🎯 Number Guessing Game

📌 About
This is a basic number guessing game written in C.
The computer generates a number between 1 and 100, and you have to guess it.

🚀 Features
Random number generation
10 attempts to guess
Hints: hot, warm, cold
Tells if you should guess higher or lower

▶️ How to Run
Compile:
gcc game.c -o game
Run:
./game

🎮 How to Play
Enter a number between 1–100
Follow the hints
Guess the correct number within 10 tries
